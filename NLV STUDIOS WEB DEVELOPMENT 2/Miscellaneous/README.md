# NLV Studios Website Development Log

---

## How to Use This Project

### 1. Getting Started
- Clone or download this repository from GitHub.
- Open the project folder.
- Launch `homepage.html` in your browser to start exploring.

### 2. Navigation Guide
The site contains seven main pages:
- **Homepage (`homepage.html`)** — Hero section, featured products, brand intro.
- **Shop Now (`shop now.html`)** — Full product listings with product cards.
- **About (`about.html`)** — Brand story, mission, and team.
- **Enquiry (`enquiry.html`)** — Form for product/sponsorship enquiries.
- **Contact (`contact.html`)** — General contact form and embedded map.
- **FAQ (`accordion.html`)** — Frequently asked questions with accordion functionality.
- **Sign up / Sign In (`register.html`)** — User registration and login form.

### 3. Features
- Responsive design with media queries for mobile, tablet, and desktop.
- Navigation bar consistent across all pages.
- Product cards styled with hover effects and pseudo‑classes.
- Forms styled with `:focus` states and validated with JavaScript.
- Accordion FAQ for user guidance.
- User registration/login functionality.

### 4. Running the Site
- No installation required — open the `.html` files directly in a browser.
